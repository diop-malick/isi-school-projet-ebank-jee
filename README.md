

# Project Creation

https://start.spring.io/

# tech stack

* Java 8
* Spring Boot 2
* Spring Security 5
* Spring MVC
* Spring Data JPA
* Thymeleaf

- MySQL (Database)

- Maven

- Bootstrap (UI Presentation)


# Install

1. Clone project

2. Create database

```sql
CREATE DATABASE `exam_jee_gestionbanque` COLLATE 'utf8_general_ci';
```

3. Import maven project on eclipse

4. Run GestionBanqueApplication.java
